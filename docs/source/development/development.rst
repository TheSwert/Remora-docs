SPI speed
---------

Using a ribbon cable the highst stable SPI speeds achieved are as follows:

+---------------+-------+--------+-----+
|               | RPi 3 | RPi 4  |     |
+---------------+-------+--------+-----+
| Core clock    | 400   | 500    | Mhz |
+---------------+-------+--------+-----+
| Clk Divider   | 64    | 64     |     |
+---------------+-------+--------+-----+
| SPI frequency | 6.25  | 7.8125 | Mhz |
+---------------+-------+--------+-----+
